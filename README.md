# FUTURE_FS_01 – Personal Portfolio Website

## Overview

A modern personal portfolio website developed as part of the Future Interns Full Stack Web Development Internship.

The portfolio showcases my skills, projects, achievements, certifications, education, and professional profile.

## Features

* Responsive Design
* Professional Developer Portfolio
* Resume Download
* Project Showcase
* Skills Section
* Achievements & Certifications
* Contact Section
* GitHub & LinkedIn Integration
* Modern UI with React and Tailwind CSS

## Tech Stack

* React.js
* Tailwind CSS
* Vite
* Framer Motion
* React Icons

## Projects Featured

1. Subscription Fatigue Tracker
2. SmartAgri AI
3. Student Analytics Dashboard
4. SaaS Billing Portal
